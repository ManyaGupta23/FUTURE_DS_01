# FUTURE_DS_01
This project analyzes business sales data to understand revenue trends, customer behavior, and product performance. The data was cleaned and processed, and key metrics such as total sales, profit, and growth trends were analyzed. Interactive dashboards and visual reports were created to help stakeholders make data-driven business decisions.
Tools Used
Power BI
Excel / CSV

Key Insights
Identified overall sales and revenue trends
Analyzed top-performing products and categories
Created interactive dashboards for easy interpretation
